About Kanbo

Any fool can create a task-list web app. Here’s mine.

Try the demonstration task grid

Project-management styles variously calling themselves Agile, Lean, Scrum, and so on inspired by Toyota’s Kanban method want to arrange cards representing stories or tasks on a board.

On the board, columns represent stages in the story’s life cycle (variously called states, bins, queues, backlogs). Stories start in the project backlog, are moved in to the In Progress bin, and eventually—we hope—end up in the Completed bin.

The board may optionally be divided in to horizontal rows to show individual projects, devlopers, or iterations (also called sprints).

Status

This is a work in progress. You can create a board of your very own and rearrange cards on it. Be warned that (a) only public boards are supported at present (anyone can read your board, if they can find it), and (b) this is still experimental code so your data might get lost during the next upgrade!
